A chatroom system developed in Winsock and imGUI.
==========

When user first open the program, there will be
a login page which ask the user to input their user
name, then pressing the ’login’ button will allow the
user enter the chatroom page, where we can see a user
list on the left panel in the window, right panel will
display all the public message the user has received, in
the bottom of the window, user can input their public
message, and press ’send’ button to send it. For the
direct message(DM), user can open the private chat
window with any other users by clicking on the user list,
where all the users besides the user himself, will show
as separate buttons. The layout of private chat window
is similar with the public window, the only difference
is that the user list won’t show in the window, and
the chat panel will only show all the DMs between
this two users. Whenever there is a user enter or exit
the chatroom, all users in the chatroom will receive a
notification, and their user list on the left will update
to the new version. Also there will be sound to remind
the user they have receive a new message, the sound
for public message and private message is different.
